== Tiger Preservation Society

Sample application for a fictional organization, which includes a forum, home, contact, and about section. The application also has the ability to sign up for an email newsletter and a contact form to send messages.

This application was made using ruby 2.2.0 and rails 4.2.0

*** This started out as learn-rails but I abandoned it for my own features at this point so it has been renamed and will be used as a custom application for my portfolio.  ***
